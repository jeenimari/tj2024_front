9. 입력 폼
    1.<input>  
        1.type 속성의 종류
            1.text   : 문자입력, 기본값 넣어도되고 안넣어도됨
            2.password : 가져온 문자 입력
            3.file : 첨부파일 입력
            4.color : 색상 입력
            5.radio : 단일선택 (name속성값이 동일)
            6.checkbox : 복수선택
            7.email  : @포함 문자입력
            8.date : 날짜입력
            9.datetime-local : 날짜 /시간 입력
            10.month : 월 입력
            11.week : 주 입력
            12.range : 범위 스크롤
            13.number : 숫자 입력
            14.submit : 폼 전송 , 추후에 자바(백엔드)와 통신 
            15.reset : 폼 초기화
            16.button : 일반 버튼 , 추후에 자바스크립트(함수,이벤트)와 연동할 수 있음.
    2.name 속성
        -각 마크업들의 식별 (구별) 가능한 이름을 정의하는곳
        -중복가능한 식별 이름

    3.value 속성 : 입력상자에 입력된 데이터
    4. readonly 속성 : 읽기모드 ,입력불가능
    5. disabled 속성 : 비활성모드 , 클릭과 수정 불가능 모드 
    6. placeholder 속성 : 입력상자의 가이드라인 텍스트
    7. maxlength 속성 : 입력상자의 최대 문자수
    8. autofocus 속성 : 지정한 입력상자의 포커스 설정 -자바스크립트와 연동할 수 있음
    

    2.<textarea> ,긴 글 텍스트상자
        -rows = "가로문자수" 행  , cols = "세로문자수 " 속성 열  , 영문기준
    
    3.<select> 목록상자
        -구성 : < select> 마크업 안에 <option> 항목 마크업을 사용함.
        <select> 
            <option>항목 1 < /option>
            <option>항목 2 < /option>
            <option>항목 3 < /option> 
    
    4.<label> , 입력상자 input에 연관된 텍스트  , 이름표 레이블이라는것자체가 이름표임
    5.<fieldset>, 폼안에서 여러 입력상자들을 묶을때(그룹) 사용
        -<legend> ,묶음의 제목/이름 설정
    
    6.<table> , 표 만드는 마크업 , 일정한 간격으로 데이터를 배치할때 쓰기도함 (아직몰라도됨)
        -<tr></tr>: 행추가 마크업
        - <th> </th? : 제목 셀(칸) 추가 마크업 굵기차이임 td랑
        - <td> </td> : 데이터 셀(칸) 추가 마크업
